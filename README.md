neural interface, and **NOXR**, a luxury e-commerce brand built on a modern full-stack

## <img src="https://media.giphy.com/media/artj92V8o75VPL7AeQ/giphy.gif" width="28"> Featured Projects

<details>
<summary><b>🧠 ZAIRE — Windows Desktop AI Assistant</b></summary>
<br/>

A native Windows AI assistant built around a plasma-orb neural HUD that reshapes itself
across five distinct modes — a general assistant, a trading copilot, a research
professor, and a full engineering agent — each with its own color identity and model
routing.

| | |
|---|---|
| **Stack** | React, Three.js, Node.js, Socket.IO, Python/Flask, ChromaDB, Groq |
| **Scale** | 5 mode-specific interfaces, per-mode LLM routing, full planning-doc suite |
| **Performance** | Real-time bidirectional streaming via Socket.IO between orchestrator and agent daemon |
| **Security** | Sandboxed QA execution with repair-diff-verify loop; dedicated security architecture doc |
| **Impact** | End-to-end designed product — from design intelligence pipeline to payments (LemonSqueezy) and auth (Clerk) |
| **Repository** | [zaire-web.netlify.app](https://zaire-web.netlify.app) |

ZAIRE's **Engineer Mode** runs a closed-loop agent that writes, tests, repairs, and
re-verifies code in a sandboxed environment, backed by schema persistence and streamed
live to the desktop UI. The Design Intelligence Layer sits between architecture approval
and scaffold generation, resolving visual identity, content strategy, and motion
decisions into a single `design_brief` object consumed downstream.

</details>

<details>

<br/>

An open-source agent orchestration framework, extracted from ZAIRE's Engineer Mode core,
designed to run entirely local-first with a companion flight-recorder-style trace
dashboard for observability.

| | |
|---|---|
| **Stack** | Python, local-first architecture, companion web Trace Dashboard |
| **Scale** | Full six-document planning suite: PRD, TRD, App Flow, UI/UX Brief, Backend Schema, Implementation Plan |
| **Performance** | Designed for local execution with no required external orchestration service |
| **Security** | Local-first by design — no mandatory cloud dependency for core agent loops |
| **Impact** | Positioned as portfolio flagship and reusable open-source infrastructure |
| **Repository** | [github.com/mughees001/ANVIL](https://github.com/mughees001/ANVIL) |

ANVIL generalizes the orchestration logic that powers ZAIRE's Engineer Mode into a
standalone, install-anywhere framework, paired with a Trace Dashboard built around an
instrument-panel / flight-recorder visual language for inspecting agent runs.

</details>
